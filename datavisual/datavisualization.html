World Happiness Report

This is the page for the data visualization made in p5.js. The data set used is the World Happiness Report from 2019, which uses a number of factors for a country's happiness score, such as GDP and generosity. For brevity, the graphic shows the top 25 and bottom 25 ranked countries on the graph. (Source)
